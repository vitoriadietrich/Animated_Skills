# Animated Skills Bar UI Design using only HTML & CSS

Um projeto simples de **barra de habilidades animada (skills bar)** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **progress bars animadas**, ideais para portfólios e páginas pessoais.

## 🛠️ Tecnologias

- HTML
- CSS (animation, keyframes)

## 🎯 Funcionalidades

- Barras de progresso animadas
- Design moderno e limpo
- Sem uso de JavaScript

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
